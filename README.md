# US Stock Data Analysis (2020–2025)
A Python data analysis project for academic course assignment.
This project uses CRSP monthly stock data from WRDS, including data cleaning, exploratory data analysis, and visualization.

## Content
- Data acquisition from WRDS database
- Data cleaning (date formatting, missing value handling, outlier removal)
- Exploratory data analysis
- Data visualization

## Charts
- Monthly return distribution
- Average stock price trend over time
- Risk‑return scatter plot

## Libraries Used
- pandas
- matplotlib
- seaborn

## Workflow
1. Data collection
2. Data cleaning
3. Data analysis
4. Visualization

## AI Use Statement
AI tools were used to assist with code generation, debugging, and result explanation. All outputs were checked, adjusted, and verified by myself.
